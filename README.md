# 🚗 CarGalaxy – Full Stack Car Listing Platform

## 📌 Project Overview

CarGalaxy is a Full Stack Web Application built using Django (Python) that allows users to browse and explore different car listings with complete details.

The application includes a fully functional Django Admin Panel where the admin can add, update, and delete cars. All changes made in the backend automatically reflect on the frontend.

This project demonstrates backend development, database integration, CRUD operations, and dynamic data handling.

## ⚙️ How It Works

1. Admin logs into Django Admin Panel
2. Adds car details (Name, Price, Description, Image)
3. Data is stored in the database
4. Frontend automatically displays updated data
5. Users can view detailed car information

## 💻 Installation Guide (Run Locally)

1️⃣ Clone the repository

git clone https://github.com/mohd-maruf-kureshi-2828/Car_Project_Using_Django.git

2️⃣ Navigate to project folder

cd Car_Project_Using_Django

3️⃣ Create virtual environment (optional)

python -m venv venv

4️⃣ Activate virtual environment

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

5️⃣ Install dependencies

pip install -r requirements.txt

6️⃣ Apply migrations

python manage.py migrate

7️⃣ Run the server

python manage.py runserver

Open in browser:
http://127.0.0.1:8000/
